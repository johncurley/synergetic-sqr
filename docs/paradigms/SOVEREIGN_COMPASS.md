# The Sovereign Compass (v1.0)
## Objective: Navigating the 16-bit Quadray Manifold via Sierpiński Fractal.

The Sovereign Compass is the user interface for the SPU-13 E-Ink display. It uses the **Sierpiński Triangle** as a map to navigate the 16-bit space of the manifold.

### 1. Reading the Triangle
Unlike a Cartesian grid, the Sovereign Compass uses **Recursive Ternary Logic**. The screen is divided into 3 primary sub-triangles (Pillars):

| Region | Meaning | SPU-13 Function |
| :--- | :--- | :--- |
| **Top Apex** | The Future | Global Davis Ratio (C) & Prediction. |
| **Bottom Left**| The Past | Dream Log & Historical Attractors. |
| **Bottom Right**| The Present | Real-time Whisper activity (The Nanos). |
| **Center Void** | The Nothing | The 'Tuck Zone' (Flashes during Henosis). |

### 2. The Sanity Map
*   **Sharp Triangles:** Manifold is **Crystalline**. High-resolution sanity.
*   **Blurry/Missing:** Manifold is **Stressed**. Lower-level attractors are evaporating.
*   **Black Center:** The system has **Tucked**. The Davis Gate is actively sealing a leak.

### 3. Navigation
The Compass 'Jitterbugs' as the SQR Rotor rotates. A 60-degree rotation shifts the bias between the three pillars, allowing the observer to see the manifold from a new perspective without losing bit-perfect integrity.
