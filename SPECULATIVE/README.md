# SPECULATIVE: Future Manifold Hypotheses

⚠️ **STATUS: Brainstorm / Unverified**

This directory contains research ideas, extrapolations, and hypotheses for future phases of the SPU-13 project. 

**None of these concepts have been physically built, measured, or peer-reviewed.**

The purpose of this folder is to document the "Ideation" phase of the project separately from the proven RTL and mathematical core. By separating speculation from fact, we maintain the technical integrity of the synthesizable SPU-13 architecture.

### Directory Contents:
- **`docs/LAMINAR_COMPONENTRY.md`**: Hypotheses on non-orthogonal physical components.
- **`docs/PHOTONICS_PLAN.md`**: Exploratory roadmap for light-based computation.
- **`docs/GEODESIC_FRACTAL_MAP.md`**: Theoretical recursive trace rules.
- **`docs/TOPOLOGICAL_ROUTING.md`**: Speculative FPGA layout strategies.
- **`docs/MINIMUM_HYSTERESIS_PROOF.md`**: Theoretical derivation of error reduction in NSE.
- **`tools/`**: Experimental scripts for visualizing or simulating these hypotheses.

### The Path Forward (Grounded):
1.  **Build the FPGA Core:** Synthesize the proven Verilog math.
2.  **Measure Reality:** Use real hardware to validate the $V_d=1.0$ invariant.
3.  **Validate Hypotheses:** Use the gathered data to determine which (if any) of these speculative ideas are supported by physical evidence.
